# π-pipe

**Тестовый стенд матричного умножения для инференса.**

Бенчмарк ядер линейной алгебры для ускорителей ИИ нового поколения. Измеряет пропускную способность умножения матриц.

## Быстрое развёртывание (для агента)

### 1. Клонирование репозитория
```bash
git clone https://github.com/dimko33-lang/pi.git
cd pi
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка окружения
Создать файл `.env` в папке проекта:
```bash
echo "GROQ_API_KEY=ВСТАВИТЬ_КЛЮЧ_ЗДЕСЬ" > .env
```

### 4. Запуск сервера
```bash
python server.py
```

### 5. Проверка работы
Открыть в браузере: `http://IP_СЕРВЕРА:8888`

---

## Структура проекта
- `index.html` — веб-интерфейс (синий минималистичный дизайн)
- `server.py` — Flask API для работы с GROQ
- `requirements.txt` — Python-зависимости
- `.env.example` — шаблон переменных окружения
